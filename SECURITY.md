# Security

## Supported versions

| Version | Supported |
|---------|-----------|
| 0.1.x   | Yes       |

## Reporting a vulnerability

Please report security issues privately via [GitHub Security Advisories](https://github.com/HN-Tran/docread/security/advisories/new) (preferred) or by opening a minimal issue asking for a contact address without posting exploit details publicly.

We will acknowledge reports as soon as possible and coordinate a fix and release.
